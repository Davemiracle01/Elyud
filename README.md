# Elyud Mapambo Website

## Jinsi ya Deploy kwenye Vercel

1. Pakia folder hii kwenye GitHub repository mpya
2. Nenda vercel.com → "Add New Project"
3. Connect GitHub repo yako
4. Framework Preset: **Other**
5. Click **Deploy** — itakuwa tayari!

## Kubadilisha Jina la Biashara
Fungua `index.html` ubadilishe "Elyud Mapambo" na jina halisi ukishalipata.
